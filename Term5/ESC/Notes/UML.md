## Unified Modelling Language
UML is extensible.
There are 14 UML diagram types.
```plantuml
@startuml
!theme toy

folder "Structural Diagrams" {
	(Composite\nStructure Diagram)
	(Deployment\nDiagram)
	(Package\nDiagram)
	(Profile\nDiagram)
	(Class\nDiagrams)
	(Object\nDiagrams)
	(Component\nDiagrams)
}

folder "Behavioral Diagrams" {
	(State Machine\nDiagrams)
	(Communication\nDiagrams)
	(Use Case\nDiagrams)
	(Activity\nDiagrams)
	(Sequence\nDiagram)
	(Timing\nDiagram)
	(Interaction\nOverview Diagram)
}

@enduml
```
