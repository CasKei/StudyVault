## Definition
> An operation $\star$ on members of a set satisfies identity, iff the set contains an element $i$, socuh that $i \star a = a$ for all $a$ in the set.

## Example
$$\mathbb{Z}^+ \text{ and } +, *$$
- $\mathbb{Z}$ and $+$ has identity 0
- $\mathbb{Z}$ and $*$ has identity 1
