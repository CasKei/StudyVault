## Pre-Sharing keys
Preshare $n(n-1)/2$ keys

**A logistical and operational nightmare**.
Assume $n=1001$:
- Keys needed: $n * (n-1)/2 \approx 1$ million
- Storage per user: $1000 * 128 bits = 16KB$
- One new user join. all $n$ users need to be updated.
